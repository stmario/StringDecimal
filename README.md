This is a small library to easily convert fixed point numbers to strings and vice versa.

Example:
```
StringDecimal.intToStringDecimal(12345, 3) = "12.345"
StringDecimal.intToStringDecimal(12345, 7) = "0.0012345"
StringDecimal.intToStringDecimal(-12345, 3) = "-12.345"
StringDecimal.intToStringDecimal(-12345, 7) = "-0.0012345"
```
